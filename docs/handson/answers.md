## 共通課題: 売上が一番高かった日はいつ？

### 解答

**2025年9月6日（土）— 売上 約8.6億円（税抜 860,198,229円）**

### 解説

この日は沖縄の旧盆最終日 **ウークイ（送り盆）** にあたります。

| 日付 | 曜日 | 売上 | イベント |
|------|------|------|---------|
| 9/3 (水) | 水 | 約7.4億 | 買い出しピーク（前夜祭的需要） |
| 9/4 (木) | 木 | 約6.1億 | ウンケー（迎え盆） |
| 9/5 (金) | 金 | 約5.4億 | ナカヌヒー（中日） |
| **9/6 (土)** | **土** | **約8.6億** | **ウークイ（送り盆）** |
| 9/7 (日) | 日 | 約4.2億 | 急落 |

### 補足

- 旧盆前日（9/3）にもウークイ当日に匹敵する買い出し需要が発生（重箱・供え物・精肉等）
- 店舗別では経塚シティ・具志川メインシティなど地域密着型の大型店舗が売上を牽引
- 旧盆期間中は地域密着型店舗の客単価が約4,000円近くまで跳ね上がる（通常時の那覇メインプレイスを逆転）
- よく売れた商品: 重箱セット、卵（サンエッグ）、シーチキン、きゅうり、バナナなど旧盆関連商品

## 課題 1: 旧盆に何が売れた？通常の何倍？

### 解答例

旧盆ウークイ当日（9/6）の売上は、通常の土曜日（8月平均）と比較して **約1.74倍** でした。

9/6の商品別売上ランキングTOP5:

| 順位 | 商品名 | 特徴 |
|------|--------|------|
| 1 | （赤）餅重箱三色セット | お供え・親戚への振る舞い用 |
| 2 | はごろもフーズ シーチキンL15P | 沖縄の常備品・行事料理の材料 |
| 3 | 助六重箱 | 仏壇へのお供え用 |
| 4 | サンエッグ 定重量玉子 | 行事料理の材料として大量消費 |
| 5 | 沖食 軽洗米 福島県産ひとめぼれギフト | お歳暮・ギフト需要 |

### 分析のヒント

- `mart_product_ranking` で全期間の売れ筋を確認
- 旧盆期間の商品別売上は `customer_sales_details` + `food_item_master` の結合で日別に確認可能
- 通常期間（例: 8月の土曜日平均）と旧盆期間を比較すると倍率が算出できる

### 補足

- 重箱セットやギフト用のお米など、旧盆特有の商品が上位を独占
- 旧盆前日には **中身汁**（豚モツの吸い物）の材料も売れやすい。沖縄の行事料理の定番
- 旧盆時期は店舗側も品目を絞って重点的に仕入れる傾向がある
- 地域密着型店舗（経塚シティ等）では客単価も通常の1.5倍以上に

## 課題 2: 曜日で売上に差はある？「日曜が一番売れる」は本当？

### 解答例

**土曜日が最も売上が高い。「日曜が一番売れる」は本当ではない。**

曜日別の平均売上（全店舗合計）:

| 順位 | 曜日 | 傾向 |
|------|------|------|
| 1 | 土曜日 | 最高（客数が平日比+約17%） |
| 2 | 日曜日 | 2位 |
| 3 | 水曜日 | 平日では最高 |
| 4 | 金曜日 | |
| 5 | 月曜日 | |
| 6 | 火曜日 | |
| 7 | 木曜日 | 最低 |

### 分析のヒント

- `mart_daily_sales` で曜日別に集計（`FORMAT_DATE('%a', PARSE_DATE('%Y%m%d', usage_date))`）
- 客数（transaction_count）と客単価（total_sales / transaction_count）を分けて分析するとより深い考察が可能

### 補足

- 土曜日に売上が高い理由の一つは **「卵の特売」**。土曜日の卵の平均単価は約207円で、平日（約256円）より大幅に安い
- 水曜日は肉の特売日（お肉3パック1,200円など）で平日トップ
- ただし旧盆（9/4木〜9/6土）を含む週は曜日パターンが大きく崩れる → 除外して分析するとより正確

## 課題 3: 一番売れている商品は？時期で変わる？

### 解答例

全期間の売上金額トップは **米・シーチキン・卵** が上位を占めます。

| 順位 | 商品名 |
|------|--------|
| 1 | 沖縄食糧 穂の舞 |
| 2 | 国産ブレンド米 |
| 3 | はごろもフーズ シーチキンL15P |
| 4 | サンエッグ さくらひめ |
| 5 | サンエッグ 定重量玉子 |

**時期で変わる？ → YES。** 8月は「穂の舞」が1位ですが、9月は「国産ブレンド米（ギフト需要含む）」が1位に浮上し、重箱セットなどもランクインします。

### 分析のヒント

- `mart_product_ranking` で全期間の売れ筋を確認（`WHERE product_name IS NOT NULL`）
- 時期別は `customer_sales_details` + `food_item_master` を結合して月別・週別に比較
- 8月 vs 9月で順位変動を比較すると旧盆の影響が見える

### 補足

- `mart_product_ranking` の `product_name` が NULL のレコードが約80%。商品名で分析する場合は `WHERE product_name IS NOT NULL` が必須
- 8月と9月で大きく変わるのは旧盆関連商品（重箱セット・ギフト用米等）の有無
- 2025年8月は **備蓄米騒動** の時期と重なっており、低価格帯のお米の売上が押し上げられている可能性がある

## 課題 4: 台風が来ると何が売れる？（8/14前後）

### 解答例

2025年8月19日に台風LINGLING（リンリン）が沖縄本島を直撃。8/19の全店売上は約3.6億円で、前後の約4億円から明確な凹みが確認できます。

**台風前日（8/18）に売れたもの:**
- 長期保存可能ですぐ食べられるもの: シーチキン、ポーク缶、食パン
- 停電・断水対策: ティッシュペーパー、牛乳、バナナ

**台風明け（8/20）に売れたもの:**
- 鮮度重視の日常食材が爆発的に売上増
- 特に **卵（ちらん赤玉たまご）** は18,000点以上の驚異的な販売数
- 台風明けに家庭での調理が再開されたことを反映

### 分析のヒント

- `mart_daily_sales` で8月中旬の日別売上推移を確認し、売上が落ち込んだ日を探す
- 課題では「8/14前後」としていますが、データを見ると台風の影響が出ているのは **8/19前後** です。「台風の日はいつか？」をデータから探し当てるプロセスも分析の醍醐味です
- 商品別は `customer_sales_details` + `food_item_master` で台風前日・当日・翌日を比較

### 補足

- 台風明けにこれだけの生鮮品が販売できているのは、サンエーの物流網が台風直後でも速やかに商品供給できていることの証拠
- 台風前の「備蓄買い」と台風後の「回復買い」で売れ筋が明確に変わるのが興味深い

## 課題 5: 旧盆が4日間ある店舗は？

### 解答例

**南部地域（那覇・浦添等）の店舗** で旧盆商戦が4日間続く傾向がある。

9/6（ウークイ）→ 9/7（翌日）の売上維持率が高い店舗:

| 店舗名 | 売上維持率 |
|--------|----------|
| パルコシティ | 146.7%（施設特性による） |
| 那覇メインプレイス | 91.5% |

南部地域の店舗は、旧盆翌日も売上がほとんど落ちていないことがデータから確認できます。

### 分析のヒント

- `mart_daily_sales` で9/4〜9/7の店舗別売上推移を確認
- 9/7（旧盆翌日）の売上落ち込みが緩やかな店舗が「4日間ある」候補
- 地域別にグルーピングすると傾向が見えやすい

### 補足

- 南部地域では旧盆翌日に「イタシキバラ（精霊払い）」などの行事が行われることが多い
- サンエー店舗の現場でも、南部地区では旧盆が実質4日間になる傾向があると認識されている
- 2025年は9/7が日曜日のため、休日需要と重なり実質的な「4日間商戦」を形成
- 他地域（北部・離島）と比較すると、南部の9/7の売上落ち込みが明らかに緩やか

## 課題 6: 営業時間が早く終わる日は？

### 解答例

**2025年9月6日（旧盆ウークイ）** — 沖縄のスーパーマーケットでは例年、ウークイの日に営業時間を短縮（20時閉店など）する慣習がある。従業員も旧盆を過ごせるようにケアするためです。

### 分析のヒント

- `shop_item_daily_performance` で日別の最終取引時間帯を確認
- ただし時間帯カラムは存在しないため、取引件数や売上金額の日内分布での推定は困難
- `mart_daily_sales` の取引数が通常より少ない日 ≠ 営業時間短縮（旧盆は客単価が高いため取引数で判断できない）

### 補足

- **注意**: このデータには時間帯の情報が含まれていないため、営業時間の短縮を定量的に証明することは困難です
- Conversational Analytics（AI）の回答は、データではなくLLMの一般知識に基づく可能性があります
- 「データだけでは分からないこともある」という気づきも、データ分析における重要な学びです

## 参考: 分析のコツ

- **まずは集計済みテーブルから**: 「日別の売上推移を見せて」のように聞くと、集計済みテーブルから回答してくれます
- **深堀りしたいときは具体的に聞く**: 「9月6日に一番売れた商品は？」のように日付を絞って質問すると精度が上がります
- **結果がおかしいと思ったら**: 期間や条件を変えて聞き直してみてください。AIの回答は毎回同じとは限りません
- **0円取引に注意**: データに異常値（0円の取引等）が含まれています。金額の集計結果に違和感があれば「0円を除外して」と追加で指示してみてください
