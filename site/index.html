About Us

沖縄から、Google Cloud で
つながる未来へ

Jagu'e'r 沖縄分科会は、2024年末に沖縄の有志によって立ち上げられた Google Cloud 公式ユーザー会の地域分科会です。

エンジニアも非エンジニアも、うちなーんちゅも県外の方も。 多様なバックグラウンドを持つメンバーが集まり、 Google Cloud を通じて沖縄の地域課題に向き合い、 テクノロジーで「うちなー」の未来を一緒につくるコミュニティです。

いつ（五）の世（四）までも
ミンサー織に込められた想いのように、
永く続くコミュニティを目指して

Mission

テクノロジーで沖縄の
地域課題を解決する

Google Cloud の先端技術を活用し、沖縄が抱える固有の課題に対して、コミュニティの力で解決策を生み出します。

Vision

沖縄を、クラウド先進
地域にする

沖縄がクラウド技術の活用においてリーディングエリアとなり、地方創生のモデルケースとして全国に発信します。

Values

ゆいまーる精神で
共に学び、共に成長する

沖縄の「ゆいまーる」（相互扶助）の精神を大切に、初心者から上級者まで誰もが気軽に参加し、助け合える場をつくります。

Activities

3つの活動の柱

沖縄分科会は3つのテーマを軸に活動を展開しています。

地域課題のDX推進

沖縄特有の産業構造や地理的課題に対して、Google Cloud の技術を活用したDXソリューションを企画・実践します。ハッカソンやプロトタイプ開発を通じ、実際の課題解決に取り組みます。

多様なコラボレーション

AI/ML分科会など他のJagu'e'r分科会との横断企画、大学との産学連携、沖縄の起業家とエンジニアのマッチングなど、多様なコラボレーションを推進します。

気軽に学べる勉強会

Google Cloud の基礎からGenerative AIの最前線まで、初心者でも安心して参加できるハンズオンや勉強会を定期開催。オンライン参加で離島からもアクセス可能です。
