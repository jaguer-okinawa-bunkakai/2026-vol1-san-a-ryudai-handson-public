SELECT 1 クエリ一覧

エージェント構築の前準備として、各テーブルに対して SELECT 1 を実行し「最近使用したアイテム」に表示させるためのクエリです。

使い方: BigQuery のクエリエディタに貼り付けて実行してください。
「全件コピー」で9テーブル分をまとめてコピーし、一括実行することもできます。

マスターテーブル

SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.card_master` LIMIT 1;
SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.customer_master` LIMIT 1;
SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.food_item_master` LIMIT 1;

トランザクションテーブル

SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.customer_sales_details` LIMIT 1;
SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.shop_item_daily_performance` LIMIT 1;

データマート

SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.mart_daily_sales` LIMIT 1;
SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.mart_customer_summary` LIMIT 1;
SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.mart_product_ranking` LIMIT 1;
SELECT 1 FROM `sana_meetup8___pos_data_analysis_dataset.mart_basket_analysis` LIMIT 1;
